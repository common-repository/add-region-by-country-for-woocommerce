Add Region by Country WooCommerce Add-on plug-in.
== Description ==

The "Add Region by Country for WooCommerce" plugin allows you to easily add and manage custom regions for specific countries within your WooCommerce store. After activation, navigate to the WooCommerce menu to configure regions through a straightforward interface. Custom regions will then be displayed on the checkout page, enhancing the shopping experience by accommodating regional variations.

Features of the plugin include:

 * Provide solution for add new region from any allowed country on WooCommerce. * 
 * Simple-configuration - just install,setup and enjoy.
 
 == Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `add-region-by-country-for-woocommerce.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the menu name WooCommerce->Add Region And here we can able to add custom Regions
4. Here we can able to see Country list and textbox for Region List. 
5. After click on this page we can see configuration form.
6. Now just configure & see the result on checkout page. 

== Frequently Asked Questions ==

= How do I activate the plugin in WordPress? =

To activate this plugin, first, go to Plugins > Add New in your WordPress Admin Dashboard. If you're using the direct install method, search for the plugin by name, click Install Now, and then Activate. Alternatively, for the upload method, click Upload Plugin, select the .zip file from your computer, click Install Now, and then Activate.

= How do I access the plugin settings? =

Go to WooCommerce > Add Region, select the country for which you want to provide custom regions, and then see the result on the checkout page.

= How do I edit already created regions? =

Go to WooCommerce > Add Region, view the list of regions, click on "Edit" for the region you want to modify, and make your changes.

= How do I remove an added regions? =

Go to WooCommerce > Add Region, view the list of regions, and click on "Edit" for the region you want to modify. From there, you can see all the regions you've added and remove the ones you no longer want.
